# library-application

** There are three different roles in the project:
  - USER;
  - MODERATOR;
  - ADMIN;
* Default: 
  - Admin:
      - username: admin;
      - password: admin;
  - Moderator:
      - username: moderator;
      - password: moderator;
  - User:
      - username: user;
      - password: user;
